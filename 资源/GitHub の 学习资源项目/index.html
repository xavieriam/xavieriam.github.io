GitHub の 学习资源项目

最后更新：2024-03-26

前言

GitHub 作为全球最大的代码托管平台，不仅拥有众多开源代码库，同时也有着许多非常有用的学习资源，将收集到的优质资源汇总于此，以便日后查阅和分享。

当然某些拓展内容包括但不限于GitHub。

文章排版

一篇优秀的文章不仅需要内容的优质，同时在排版上也需要美观（谁叫我们是视觉动物？）。

  • 中文文案排版指北

    「详细全面地讲解了文案排版的风格规范。」

  • 中文技术文档的写作规范

    「阮一峰老师所著，内容同样非常全面。」

  • 少数派写作排版指南

    「这是少数派文书表述上的一些文字格式和风格规范。」

  • Butterick’s Practical Typography

    「这个网站介绍了关于排版和字体的相关知识，侧重使用和在印刷，网站等产品上的实际体验。」

如何提问

如何高效提问是一个重要命题。

  • 提问的智慧

    「在互联网上如何抛出一个问题，并获得你想得到的答案？这本书告诉了你应该如何去做。」

  • 学会提问

    「此为豆瓣书链。《学会提问》是一本适用范围更广泛的书籍，推荐阅读。」
